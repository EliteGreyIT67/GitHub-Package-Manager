## quick install :part_alternation_mark: :computer:

### MacOS / Linux *v1.5.x*
	git clone https://github.com/smoke-wolf/GitHub-Package-Manager.git
	cd GitHub-Package-Manager
	python3 Start.py
	
Post-Account notes:

    python3 CLI.py
Then Select **Option 4** From The List.
And Finaly: 

    python3 Start.py
